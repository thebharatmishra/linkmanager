# Smart Link Manager

A clean, modern link manager that helps you organize and manage your links efficiently.

## Features

- **Smart Parsing**: Automatically detects URLs and titles in "URL | Title" format
- **Auto-grouping**: Links are automatically categorized by domain
- **Rich Preview**: Shows favicons, descriptions, and tags
- **Search & Filter**: Find links quickly with advanced search
- **Bulk Actions**: Move, delete, or group multiple links at once
- **Import/Export**: Backup and restore your links as JSON files
- **Keyboard Shortcuts**: Quick actions with keyboard shortcuts
- **Responsive Design**: Works on desktop and mobile

## How to Use

1. **Save Links**: Paste your links in the textarea (one per line)

   - Format: `URL | Title` (optional)
   - Example: `https://youtube.com/watch?v=abc123 | Cool Video`

2. **Organize**:

   - Links are automatically grouped by domain
   - Create custom groups with colors
   - Drag and drop to reorganize

3. **Find Links**:

   - Use the search bar to find links
   - Filter by group or sort by date/title

4. **Manage**:
   - Click "Open" to open links in new tabs
   - Click "Delete" to remove links
   - Use bulk actions for multiple links

## Keyboard Shortcuts

- `Ctrl + Enter`: Save links
- `Ctrl + S`: Save links
- `Ctrl + F`: Focus search
- `Escape`: Close modals

## Storage

All data is saved in your browser's local storage. Export your links regularly for backup.

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
